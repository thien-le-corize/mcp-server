---
description: Kiểm tra nhanh sức khỏe server - dùng MCP server ttmkt
---

Dùng MCP server `ttmkt` để kiểm tra nhanh:

1. `get_memory_usage` - RAM/Swap
2. `get_disk_usage` - Disk
3. `get_pm2_status` - PM2 processes
4. `get_top_processes(sort="cpu", count=10)` - Top CPU

Trả kết quả dạng bảng tóm tắt, highlight vấn đề (nếu có). Tiếng Việt.
