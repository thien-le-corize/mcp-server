═══ BLOCKERS ═══

Several tools were unavailable, limiting analysis depth:

- **sar (sysstat)**: Not configured - cannot read /var/log/sysstat/sa23
  - Missing: CPU usage history, load average timeline, memory pressure
  - Impact: Cannot confirm CPU saturation during incident
  
- **dmesg**: Permission denied for mcp-reader user
  - Missing: OOM killer events, kernel panics
  
- **PHP error log**: Permission denied /var/log/php8.0-fpm.log
  - Missing: PHP fatal errors, timeouts during incident

Fix:
```bash
# Install and enable sysstat
sudo apt install sysstat
sudo systemctl enable sysstat
sudo systemctl start sysstat

# Grant read access to system logs
sudo setfacl -m u:mcp-reader:r /var/log/php8.0-fpm.log
sudo setfacl -m u:mcp-reader:rx /var/log/sysstat
```
